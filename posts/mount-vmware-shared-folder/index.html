在 Ubuntu 中 mount VMware 共享文件夹

Created2019-5-22|Linux
|Word count:96|Reading time:1min

1. 安装 VMware Tools

  • 给虚拟机增加一个光驱

  • VM> Install VMware Tools

  • 手动安装

  • 可以删掉光驱

2. 映射共享文件夹

  • 查看 /mnt/hgfs 目录下是否已有映射的目录
  • 如果没有，/usr/bin/vmware-config-tools.pl
