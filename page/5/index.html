Windows 8.1 USB 3.0 速度异常的处理
Created2015-9-6|Windows|Windows 8.1•USB 3.0•Intel
Windows 8.1 已经内建 USB 3.0 控制器驱动，正常情况下无需单独安装 USB 3.0 驱动程序。但有时由于兼容性问题或电源策略问题，系统睡眠并重新唤醒后 USB 3.0 端口无法正常工作，现象为 USB 3.0 设备传输速度限制在 30MB/s 上下，系统识别端口为 USB 2.0 端口。如图，设备和打印机中查看 3.0 设备时提示可在 3.0 端口上获得更快速度。 首先强制重新安装驱动。在英特尔官网下载最新版本驱动程序。此时若直接安装会提示系统版本不正确，无法安装。进入 \Drivers\Win7 下与系统版本对应（32/64位）的目录，右键 iusb3hub.inf 和 iusb3xhc.inf 安装。在设备管理器中双击 USB 设备中最下面的 USB Root Hub (xHCI)，在电源管理选项卡中取消选择“允许计算机关闭此设备以节约电力”。 重启电脑。在设备和打印机中重新查看设备工作状态，提示已插入 3.0 端口。测速 200+MB/s。 休眠并唤醒后不变。Done. by Sykie Chen 2015.9.6
